IoW has a DEBUG MODE to help testers with debugging
The purpose of the DEBUG MODE is to allow the testers to move freely trough the game to optimize their testing sessions

Press ENTER to toggle DEBUG MODE

When the mode is on :
-WASD keys to Move freely through the air and through walls



in non debug mode the controls are as follows:

W - jump
A - move left
D - move right


Spacebar  - melee attack

Left mousebutton - ranged attack

Left shift - sliding

ESC - to open menu
